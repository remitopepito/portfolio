Projet mené en groupe de 3 - avril 2024

Créer une base de données fonctionnelle.

Quel était l'objectif de notre travail ?

Les objectifs de ce projet étaientt de créer une base de données, l’alimenter à l’aide de fichiers CSV, puis créer une application avec un affichage TKinter pour interagir avec la base.

Comment avons-nous procédé ?

Nous avons procédé par étapes pour créer la base. Nous avons commencé générer le script SQL pour créer les tables dans la base, après avoir lancé le serveur. Nous avons ensuite créé un programme Python qui permettait d’extraire les données des fichiers CSV, de faire des jointures entre eux, de changer le format de certaines données, afin de les charger dans la base. Enfin, nous avons développé le Tkinter. Pour finir, nous avons rédigé un compte-rendu de notre travail dans lequel nous devions détailler notre projet.

Quelles difficultés avons-nous rencontrées ?

Personnellement, j’ai surtout travaillé sur le système de chargement automatique des données. J’ai rencontré pas mal de difficultés, comme le fait de devoir réorganiser les données, car elles étaient disposées différemment entre les fichiers CSV et les tables dans la base. J’ai aussi eu des problèmes concernant la lecture des fichiers CSV, car je devais toujours spécifier le chemin pour y accéder. J’ai donc créer un système pour que l’utilisateur puisse spécifier le bon chemin.

Quelle note avons-nous obtenue ?

Nous avons obtenu 14/20.

Quelles sont mes compétences acquises ?

Personnellement, cette SAE m'a permis de progresser en Python. J'ai pu aussi prendre en main l'environnement de phpMyAdmin. J'ai pu mettre en pratique mes connaissances en SQL, et je pense que je me suis amélioré sur ce langage.
