Projet mené en binôme

Prédire avec précision les valeurs immobilières de la ville de Paris.

Quel était l'objectif de notre travail ?

Nous étions chargés de prédire les valeurs immobilières des logements de Paris à l’aide d’un modèle statistique à partir de variables comme la surface habitable ou bien l’arrondissement. À la fin, nous devions fournir le fichier CSV avec les prédictions finales, le code R qui a permis de le générer ainsi qu’un rapport sur le travail que nous avions fournis.

Comment avons-nous procédé ?

Tout d’abord, nous avons commencé par analyser les données sur Excel à l’aide d’un premier fichier qui contenait les valeurs. Nous avons déterminé la meilleure méthode à utiliser pour effectuer nos calculs de prédictions, puis nous les avons appliqués avec R sur un premier jeu de données où nous avons pu comparer nos prédictions aux valeurs réelles. Nous avons corrigé notre modèle, puis nous l'avons appliqué à un deuxième jeu pour y prédire les valeurs immobilières. Pour terminer, nous avons rédigé un rapport qui rendait compte du travail fourni.

Quelles difficultés avons-nous rencontrées ?

L’élaboration de notre modèle statistique a été la tâche la plus difficile que nous avons rencontré. Nous n’avons pas eu de problèmes concernant le code R et la rédaction du document de synthèse.

Quelle note avons-nous obtenue ?

Notre travail a obtenu la note de 15,5/20.

Quelles sont mes compétences acquises ?

Ce projet m'a permis d'appliquer les méthodes de calculs vues en cours de régression linéaire, avec le langage R. J'ai donc aussi progressé en R.
