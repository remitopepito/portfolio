Projet mené en binôme - décembre 2023

Trier. Ordonner. Convertir. Rendre exploitables les données qui nous sont fournies.

Quel était l'objectif de notre travail ?

Nous disposions d'un fichier au format JSON, un format difficile à lire, qui contenait des données concernant des comorbidités de différentes pathologies. Afin de rendre le fichier exploitable, nous avons été chargés de créer un programme Python capable de le trier et de le convertir au format CSV pour pouvoir le lire sur Microsoft Excel.

Comment avons-nous procédé ?

Pour commencer, nous avons dû trouver le moyen de lire le fichier JSON avec le code Python. Ensuite, nous avons créé une boucle qui parcourait le fichier et qui alimentait une grande liste. Cette liste était ensuite transformée en fichier CSV, avec les normes spécifiées dans les consignes.

Quelles difficultés avons-nous rencontrées ?

La difficulté majeure a été de mettre au point la boucle dans le programme qui triait les données. Nous avons passé beaucoup de temps à la faire fonctionner. Néanmoins, nous avions une certaine avance par rapport au temps imparti pour terminer le projet, donc cela ne nous a pas nécessairement pénalisé.

Quelle note avons-nous obtenue ?

Notre travail a obtenu la note de 14/20.

Quelles sont mes compétences acquises ?

Cette SAE m'a permis de mieux comprendre l'utilité des listes et des dictionnaires sur Python. Plus globalement, j'ai appris à manipuler un fichier avec un format particulier pour le nettoyer et le trier avec de le rendre lisible et exploitable dans un format adéquat. J'ai donc progressé en Python.
